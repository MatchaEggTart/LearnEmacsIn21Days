# 界面设置

* 菜单栏关闭
  * HOW

    ``` lisp
    ;; 菜单栏关闭
    (menu-bar-mode -1)
    ```

* 工具栏关闭
  * HOW

    ``` lisp
    ;; 工具栏关闭
    (tool-bar-mode -1)
    ```

* 滚动条关闭
  * HOW

    ``` lisp
    ;; 滚动条关闭
    (scroll-bar-mode -1)
    ```

* 关闭启动界面
  * HOW

    ``` lisp
    ;; 关闭启动界面
    (setq inhibit-startup-screen t)
    ```
